Role Name
=========

This role can be used to install cloud endure agent on different linux flavors.

More on CloudEndure can be seen [here](https://www.cloudendure.com/)
Requirements
------------

 MakeSure you have CloudEndure Agent Installtion Keys.

Execution
------------

      roles:
